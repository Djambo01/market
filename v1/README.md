# Интернет-магазин с системой рекомендаций (v1)

## Описание проекта

Базовая версия приложения интернет-магазина с простой системой рекомендаций товаров на JavaScript.

## Функциональность

### Товары
- Каталог товаров с информацией:
  - ID товара
  - Название
  - Цена
  - Категория
  - Рейтинг
  - Популярность

### Система рекомендаций (v1)
В этой версии реализованы следующие типы рекомендаций:

1. **Популярные товары** - товары отсортированы по параметру популярности
2. **Товары с высоким рейтингом** - товары с лучшей оценкой пользователей
3. **Случайные рекомендации** - произвольный выбор товаров из каталога
4. **Товары по категориям** - фильтрация товаров по разделам магазина

## Файлы проекта

- `products.js` - база данных товаров
- `recommendations.js` - функции рекомендаций
- `index.js` - главный файл приложения
- `README.md` - документация

## Использование

```bash
node index.js
```

## Версии проекта

- **v1** - текущая версия (базовая функциональность)
- **v2** - планируется: расширенная система рекомендаций с историей пользователя
