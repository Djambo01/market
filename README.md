# 🛒 Интернет-магазин с системой рекомендаций

Проект демонстрирует эволюцию системы рекомендаций товаров на JavaScript с использованием Git для версионирования.

## 📂 Структура проекта

```
ars/
├── v1/                          # Базовая версия
│   ├── products.js              # Каталог товаров
│   ├── recommendations.js       # Простые рекомендации
│   ├── index.js                 # Главный файл
│   └── README.md                # Документация v1
│
├── v2/                          # Расширенная версия
│   ├── products.js              # Каталог товаров (тот же)
│   ├── recommendations.js       # Расширенные рекомендации
│   ├── index.js                 # Главный файл
│   └── README.md                # Документация v2
│
├── .gitignore                   # Исключения для Git
├── README.md                    # Этот файл
├── GIT_COMMANDS.md              # Справка по Git командам
└── PREPARE_FOR_SUBMISSION.md    # Подготовка к сдаче
```

## 🎯 Что в проекте

### ✅ Версия v1 - Базовая функциональность

**Функции:**
- Список товаров с характеристиками (название, цена, рейтинг, популярность)
- Простые рекомендации:
  - Популярные товары
  - Товары с высоким рейтингом
  - Случайные товары
  - Фильтрация по категориям

**Запуск:**
```bash
node v1/index.js
```

### 🚀 Версия v2 - Расширенная функциональность

**Новые возможности:**
- Класс `UserHistory` для отслеживания действий пользователя
- Умные рекомендации на основе истории просмотров
- Исключение уже просмотренных товаров
- Анализ предпочитаемых категорий
- Рекомендации непросмотренных товаров

**Запуск:**
```bash
node v2/index.js
```

## 🔍 Как понять, как работают рекомендации

### v1 - Базовая логика
```javascript
// Популярные товары - сортируем по полю popularity
getPopularProducts = () => 
  products.sort((a,b) => b.popularity - a.popularity)
```

### v2 - Умные рекомендации
```javascript
// Анализируем историю пользователя
// Исключаем просмотренные товары
// Приоритет товарам из предпочитаемых категорий
getSmartRecommendations = (history) => {
  const preferred = history.getPreferredCategories()
  return products
    .filter(p => !history.viewedProducts.includes(p.id))
    .filter(p => preferred.includes(p.category))
    .sort((a,b) => b.rating - a.rating)
}
```
