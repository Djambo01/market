# Интернет-магазин с системой рекомендаций (v2)

## Описание проекта

Улучшенная версия приложения интернет-магазина с интеллектуальной системой рекомендаций, учитывающей историю просмотров пользователя.

## Новые возможности в v2

### Расширенная система рекомендаций
1. **Умные рекомендации** - анализируют историю просмотров и предлагают товары на основе предпочтений
2. **Исключение просмотренных товаров** - уже просмотренные товары не рекомендуются повторно
3. **Товары из предпочитаемых категорий** - система запоминает, какие категории интересуют пользователя
4. **Непросмотренные товары** - популярные товары, которые пользователь ещё не видел

### Класс UserHistory
Отслеживает историю действий пользователя:
- Просмотренные товары
- Предпочитаемые категории
- Статистика по категориям

## Файлы проекта

- `products.js` - база данных товаров
- `recommendations.js` - классы и функции рекомендаций
- `index.js` - главный файл приложения
- `README.md` - документация

## Использование

```bash
node index.js
```
